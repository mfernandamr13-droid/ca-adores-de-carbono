# caçadores-de-carbono
Caçadores de Carbono é um jogo educativo desenvolvido para uma Feira de Ciências, com o objetivo de ensinar de forma interativa sobre o papel do carbono e do CO₂ no meio ambiente.
No jogo, o jogador controla um cientista que precisa capturar moléculas de CO₂ antes que elas cheguem à atmosfera, aprendendo ao mesmo tempo sobre diferentes biomas e sua importância no armazenamento de carbono.

🎮 Sobre o jogo:
O jogo foi criado usando HTML, CSS e JavaScript, utilizando um canvas para renderizar os elementos do jogo.
Durante a partida, o jogador precisa:
* Capturar moléculas de CO₂
* Completar níveis baseados em diferentes biomas
* Aprender curiosidades científicas sobre o meio ambiente
* Ajudar a reduzir o impacto do carbono na atmosfera

## 🌱 Objetivo educativo:
Este projeto foi desenvolvido para:
* Explicar o **ciclo do carbono**
* Mostrar a importância das **florestas e biomas**
* Conscientizar sobre **mudanças climáticas**
* Tornar o aprendizado de ciência **mais divertido e interativo**

## 🕹️ Como jogar:
1. Abra o arquivo **index.html** no navegador
2. Clique em **Começar Jogo**
3. Capture o máximo de **CO₂** possível antes que o tempo acabe
4. Avance pelos biomas e descubra curiosidades científicas
